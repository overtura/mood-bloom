# 제품 brief

Mood Bloom은 사용자가 하루에 한 문장을 입력하면, 브라우저 로컬 모델이 문장에서 받은 인상을 제한된 `MoodDecision`으로 분류하고 결정적 생성 엔진이 디지털 식물과 개인 정원으로 표현하는 앱이다.

## 핵심 사용자 흐름

```text
오늘의 한 문장
→ 브라우저 로컬 의미 분류
→ speciesSeed와 dailySeed
→ 씨앗 발아와 오늘의 식물 reveal
→ Future Bloom Day 1/3/7/14/30
→ 개인 정원 저장
→ 누적 고유 기록일에 따른 정원 진화
```

첫 사용의 목표는 긴 설명을 읽게 하는 것이 아니라 첫 식물을 피운 직후 성장 가능성을 직접 만져보게 하는 것이다. Future Bloom은 실제 상태를 바꾸지 않으며 “가능한 성장 예시”임을 항상 밝힌다.

## 사용자 약속

- 하루를 쉬어도 기존 진행이나 해금을 잃지 않는다.
- streak는 장식적인 표현일 뿐 핵심 성장 조건이 아니다.
- 죄책감, 긴급성, 손실 회피를 이용하지 않는다.
- 문장의 인상을 빛, 움직임, 방향, 형태로 설명하고 심리를 단정하지 않는다.
- 문장과 기록은 사용자의 브라우저를 떠나지 않는다.

## 비범위

- backend, 계정, 동기화, 소셜 피드, 결제, push 알림
- 외부 AI 추론 API와 자유 텍스트 생성
- 감정 확률 노출, 심리 검사, 의료적 해석
- 모델 binary의 저장소 포함
